# HTML

## Review

- HTML
- Markup language
- Programing language
- HTML structure
  - `<!DOCTYPE html>`
  - `<html></html>`
  - `<head></head>`
  - `<body></body>`
- HTML syntax
  - elements
    - opening tag
      - attributes
        - name
        - value
    - content
    - closing tag
- tags
  - `<h1>` - `<h6>`
  - `<p>`
  - `<span>`
  - `<a>`
    - `href=""`
  - `<img src="" alt="">`
  - `<ul>`
    - `<li>`
  - `<ol>`
    - `<li>`
  - `<dl>`
    - `<dt>`
    - `<dd>`
  - `<div>`
  - `<style>`
- id
- class
