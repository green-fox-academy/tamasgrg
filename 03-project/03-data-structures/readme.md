## Material Review

- ArrayLists
  - Object
  - means to *add* and *remove*
  - initialize
  - `.add()` (to the next position, at an index)
  - `.addAll()`
  - `.remove()` (by index, by value)
  - `.set()`
  - `.size()`
  - `.isEmpty()`
  - `.clear()`
  - searching: `.contains()`, `.containsAll()`
  - assigning between arrays (*copying* `.toArray()`, `.addAll()`)
  - casting as string `.toString()`
- String
  - `.length()`
  - `.substring()`
  - `.replace()`
  - `.charAt()`
  - `.equals()`
  - `.compareTo()`
  - `.toLowerCase()`
  - `.contains()`
  - `.endsWith()`
  - `.startsWith()`
  - `.indexOf()`
  - `.lastIndexOf()`
  - `.split()`
- HashMap
  - key
  - value
  - `.put()`
  - `.get()`
